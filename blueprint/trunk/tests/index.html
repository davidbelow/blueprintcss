Blueprint CSS - Sample and Test Documents

Follow the links below to view the sample and test documents for the Blueprint CSS Framework. These documents are designed to display the features of the framework and give you a starting point for writing your own grid based pages. All of these test files are included with the Blueprint CSS framework and can be found in the 'tests' sub-directory.

  • sample.html - A sample layout designed to show some of what is possible with a grid layout.
  • grid.html - Shows some of what is possible with a grid layout.
  • typography.html - Shows some of the typographic enhancements you get with use of Blueprint CSS.

We hope you enjoy using Blueprint CSS.
